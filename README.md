# CALCULADORA IMC

Aplicação console para cálculo da área e do perimetro do quadrado e do retângulo.

## Funcionalidades

- Calcula a Área.
- Calcula o Perimetro.

## Tecnologias utilizadas
 
- dotnet 6.0

## Como executar
 Realize o clone do repositório e execute o comando:
 
 ```
 dotnet run 
 ```
